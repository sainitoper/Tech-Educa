# Tech-Educa
This is a full-stack education website that allows users to explore available courses and send messages through a contact form. The project includes both frontend and backend components and is built using modern web technologies including **HTML, CSS, JavaScript, Java (Spring Boot), Hibernate, and MySQL**.  ---
---

## 📌 Features

- 🏠 Homepage with clean UI
- 📖 About section
- 📚 Dynamic course listing fetched from database
- 📩 Contact form that stores user messages in the backend
- 🔒 Secure data handling with proper validations

---

## 🧱 Tech Stack

### Frontend
- **HTML** – Page structure
- **CSS** – Styling and layout
- **JavaScript** – Basic interactivity and form handling

### Backend
- **Java** with **Spring Boot** – REST APIs and application logic
- **Hibernate** – ORM to interact with the database
- **MySQL** – Database to store courses and contact messages
